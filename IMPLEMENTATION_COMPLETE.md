## 🚀 Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt requirements-streamlit.txt

# 2. Configure
cp .env.example .env
# Edit .env with ANTHROPIC_API_KEY

# 3. Validate
python3 validate_demo.py

# 4. Test (optional)
pytest tests/test_streamlit_demo.py -v

# 5. Launch
./run_demo.sh
```
